# Hero Lab Implementation Plan

> **For agentic workers:** Execute inline in this session. No commits (user constraint). No test framework — verify with `npm run build` and manual `/lab` check.

**Goal:** Ship `/lab` with five full-viewport hero variations sharing one personal pitch, without touching `/`.

**Architecture:** Dedicated `LabPage` stacks five hero components. Shared copy in `sharedPitch.ts`. Each hero owns its palette + motion (GSAP / Three / Framer). Lab chrome: dots nav + evaluation labels. Animate only when in view; honor `prefers-reduced-motion`.

**Tech Stack:** React 19, Vite, Tailwind v4, GSAP + `@gsap/react`, Three.js, Framer Motion (existing).

## Global Constraints

- Do not modify root `/` or `ProfileCard`
- Branch `feat/hero-lab`; **do not commit**
- Path: `/lab`
- Same pitch in every hero (brand + headline + support + CTAs)
- New palettes per hero; not cream/terracotta identity of root
- `prefers-reduced-motion` → static finals
- Pause Three when off-screen; cap DPR ≤ 1.5
